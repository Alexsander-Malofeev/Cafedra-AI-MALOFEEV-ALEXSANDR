# HW06 – Report

## 1. Dataset

**Какой датасет выбран:** S06-hw-dataset-04.csv

**Размер:** 10000 строк, 21 столбец (20 признаков + целевая переменная)

**Целевая переменная:** target (бинарная классификация)
- Класс 0: 9500 записей (95.00%)
- Класс 1: 500 записей (5.00%)

**Признаки:** 
- Все признаки числовые (float64)
- Категориальных признаков нет
- Пропущенных значений не обнаружено
- Признаки имеют различный масштаб, требуется масштабирование для линейных моделей

## 2. Protocol

**Разбиение:** train/test
- Доля тестовой выборки: 25% (2500 записей)
- random_state: 42
- Стратификация по целевой переменной применена для сохранения баланса классов

**Подбор гиперпараметров:** CV на train
- Количество фолдов: 5
- Стратифицированные фолды (StratifiedKFold)
- Оптимизируемая метрика: ROC-AUC
- GridSearchCV для поиска оптимальных параметров

**Метрики:**
- **Accuracy:** Базовая метрика, но может вводить в заблуждение из-за дисбаланса
- **F1-score:** Баланс между precision и recall, важна для задач с дисбалансом
- **ROC-AUC:** Основная метрика для сравнения моделей, устойчива к дисбалансу классов, показывает качество разделения классов

## 3. Models

**Сравниваемые модели:**

1. **DummyClassifier (baseline)**
   - Стратегия: 'most_frequent'
   - Всегда предсказывает наиболее частый класс (0)

2. **LogisticRegression (baseline из S05)**
   - Pipeline со StandardScaler
   - class_weight='balanced' для учета дисбаланса
   - max_iter=1000

3. **DecisionTreeClassifier**
   - Контроль сложности через max_depth и min_samples_leaf
   - Подбираемые параметры:
     - max_depth: [3, 5, 7, 10]
     - min_samples_leaf: [1, 5, 10]
     - criterion: ['gini', 'entropy']

4. **RandomForestClassifier**
   - Ансамбль деревьев с bagging и случайностью признаков
   - Подбираемые параметры:
     - n_estimators: [50, 100]
     - max_depth: [5, 10]
     - min_samples_leaf: [1, 3]
     - max_features: ['sqrt']

5. **GradientBoostingClassifier**
   - Последовательный boosting алгоритм
   - Подбираемые параметры:
     - n_estimators: [50, 100]
     - learning_rate: [0.1, 0.2]
     - max_depth: [3, 5]
     - subsample: [0.8]
